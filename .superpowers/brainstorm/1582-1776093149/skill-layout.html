方案 A 优点

  • 卡片展示更直观，信息丰富
  • 直接显示已分配角色标签
  • 弹窗编辑快速，无需跳转
  • 适合少量 Skill 管理

方案 A 缺点

  • 大量 Skill 时卡片列表较长
  • 弹窗空间有限，编辑长内容不便

方案 B 优点

  • 表格紧凑，适合大量数据
  • 独立详情页编辑空间大
  • Monaco Editor 全屏体验好
  • 与现有角色管理页面风格一致

方案 B 缺点

  • 需要跳转才能看到详情
  • 操作步骤稍多
