弹窗编辑功能说明

  • Skill ID：创建时必填，编辑时不可修改（灰色显示）
  • 显示名称：可选，用于友好展示
  • 描述：简短说明，显示在列表卡片中
  • 分配角色：勾选列表，已选角色高亮显示
  • Markdown 编辑器：Monaco Editor，支持语法高亮、行号、自动补全
  • 预览按钮：可切换查看 Markdown 渲染效果
  • 弹窗尺寸：宽度约 700px，高度自适应或最大 80vh
