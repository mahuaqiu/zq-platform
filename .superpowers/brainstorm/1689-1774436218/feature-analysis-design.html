设计规范说明

  • 筛选区域：灰色背景 #fafafa，每个筛选项有标签，统一上下结构
  • 表格：带边框，边框颜色 #e8e8e8，字体 13px，padding 12px 10px
  • 对齐规则：
    • 文本类列（编号、名称、责任人等）：标题和内容都居左
    • 数值类列（数量、状态、百分比等）：标题和内容都居中
  • 编号类字段：使用 code 样式，背景 #f5f5f5
  • 状态颜色：成功/良好 #52c41a、警告/进行中 #faad14、错误/严重 #ff4d4f、信息 #1890ff
  • 统计卡片：特性进度页面顶部增加统计概览
