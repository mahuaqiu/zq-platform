响应式设计

大屏（≥1200px）
- 左侧曲线图：65%
- 右侧栏：35%
- 次要指标：2列网格
- 曲线图高度：CPU 160px，其他 120px
中屏（768-1199px）
- 左侧曲线图：60%
- 右侧栏：40%
- 次要指标：2列网格
- 曲线图高度：统一 100px
小屏（<768px）
- 单列布局
- 曲线图在上，次要指标在下
- 次要指标：1列
- TOP10 可折叠

数据流向

实时采集：
Worker → POST /report → 后端存储 → 前端轮询 GET /latest → 更新曲线图

历史查看：
用户选择时间窗口 → GET /data → 后端返回历史数据 → 曲线图渲染

版本对比：
用户选择版本 → GET /compare → 后端返回多版本数据 → 曲线叠加显示
布局要点：
  • 顶部固定控制栏，便于快速操作
  • 时间轴选择器独立一行，突出时间控制
  • 左侧曲线图占主体，右侧补充次要信息
  • 次要指标紧凑排列，不占用过多空间
  • TOP10 概览提供快速参考，详情可在 Tooltip 查看
