# 版本对比页面重新设计规格文档

> 创建日期: 2026-05-17
> 作者: Claude Code

## 1. 概述

### 1.1 背景

性能监控页面已全面改版，版本对比页面需要重新设计以保持风格统一。同时修复版本选择不到的问题。

### 1.2 问题分析

**问题：标记版本在版本对比页面选择不到**

原因分析：版本对比页面从路由参数获取 `device_id`，如果标记版本时使用的是设备 A，但进入版本对比页面时没有正确传递设备 ID 参数，就会获取不到标记的版本。

**解决方案**：版本对比页面移除设备选择，直接获取所有标记的版本进行对比，不关心设备来源。

### 1.3 目标

- 与性能监控新设计风格保持一致
- 移除设备选择，简化版本对比流程
- 支持更多 HWiNFO 指标对比
- 统一标签/标记弹窗设计风格
- 优化数据摘要展示

### 2.1 整体布局结构

```
┌─────────────────────────────────────────────────────────┐
│ [版本对比标题] [版本选择器卡片] [开始对比] [导出报告]    │
├─────────────────────────────────────────────────────────┤
│ [指标选择器：CPU GPU 内存 提交内存 更多指标▼]           │
├─────────────────────────────────────────────────────────┤
│ [时间导航条：时间范围 | dataZoom | 快速按钮 | 标签列表] │
├─────────────────────────────────────────────────────────┤
│ [主图表区域 - 多版本曲线叠加 + 标签区间可视化]          │
│ [底部 dataZoom 缩放条]                                  │
├─────────────────────────────────────────────────────────┤
│ [数据摘要表格] │ [版本进程详情面板]                     │
└─────────────────────────────────────────────────────────┘
```

### 2.2 顶部控制栏（标题 + 版本选择器同行）

**位置**: 页面顶部

**组成**:
- 左侧：标题"版本对比"
- 中间：版本选择器卡片（已选版本标签 + "添加版本"按钮）
- 右侧：操作按钮（开始对比、导出报告）

**版本选择器卡片样式**:
- 已选版本：彩色背景（#409eff、#67c23a、#e6a23c等），白色文字，圆角 6px
- 删除按钮：右上角红色圆形 × 按钮
- "添加版本"按钮：白色背景，蓝色虚线边框，蓝色文字

**交互**:
- 点击版本卡片上的 × 删除该版本
- 点击"添加版本"打开版本选择下拉框
- 最多选择 6 个版本

### 2.3 指标选择器

**位置**: 顶部控制栏下方

**组成**:
- 固定显示指标卡片：CPU、GPU、内存、提交内存
- "更多指标 ▼"按钮

**样式**:
- 选中的指标：蓝色背景（#409eff），白色文字，圆角 6px
- 未选中的指标：白色背景，蓝色边框（2px solid），蓝色文字

**更多指标弹窗**:
- 与性能监控页面一致
- 支持搜索 HWiNFO 200+ 传感器指标
- 显示中文名称 + 英文 KEY

### 2.4 时间导航条

**位置**: 指标选择器下方

**组成**:
- 左侧：时间范围显示（YYYY-MM-DD HH:MM ~ HH:MM）
- 中间：ECharts dataZoom 导航条（25%宽度）
- 快速选择按钮：15分钟、60分钟、12小时
- 右侧：标签列表 + "添加标签"按钮

**dataZoom 样式**（与性能监控页面一致）:
- 高度：18px
- 背景色：#e8e8e8
- 选中区域：rgba(64, 158, 255, 0.15)
- 把手：白色背景，蓝色边框

**标签显示**:
- 冲高标签：红色边框 + 红色文字 (#f56c6c)
- 稳态标签：绿色边框 + 绿色文字 (#67c23a)
- 显示格式：标签名称（类型）× 删除按钮

### 2.5 主图表区域

**位置**: 时间导航条下方

**图表类型**: ECharts 折线图 + dataZoom

**数据系列**:
- 多版本曲线叠加，每个版本使用不同颜色
- 颜色方案：#409eff（蓝）、#67c23a（绿）、#e6a23c（橙）、#f56c6c（红）、#9b59b6（紫）、#909399（灰）

**标签区间可视化**:
- 冲高区间：红色背景遮罩
- 稳态区间：绿色背景遮罩

**图表高度**: 约 240px

**底部缩放条**:
- ECharts 内置 dataZoom slider
- 支持拖动缩放时间范围

### 2.6 数据摘要表格

**位置**: 主图表下方左侧

**标题**: "数据摘要（标签区间统计）"

**列定义**:
- 版本名称（带颜色）
- 系统CPU、进程CPU、GPU、提交内存（根据当前指标调整）

**数值显示**:
- 最优值：绿色 ✓ 标记
- 最差值：红色 ✗ 标记

**统计逻辑**: 合并所有标签的时间区间，计算区间内的统计值

### 2.7 版本进程详情面板

**位置**: 数据摘要右侧，宽度 320px

**标题**: "版本进程详情（悬停显示）"

**内容**:
- 每个版本的进程详情卡片
- 左侧彩色边框标识版本颜色
- 显示进程名、PID、当前指标值

**交互**: 悬停图表时同步更新各版本进程数据

## 3. 标签功能设计

### 3.1 标签类型

- **冲高**: 用于标记峰值区间，红色显示
- **稳态**: 用于标记稳定区间，绿色显示

### 3.2 添加标签弹窗

**弹窗宽度**: 400px

**表单字段**:

| 字段 | 类型 | 说明 |
|------|------|------|
| 标签名称 | 文本输入 | 如：场景加载、发起共享 |
| 区间类型 | 下拉选择 | 冲高 / 稳态 |
| 开始时间 | 日期时间选择器 | 绝对时间 |
| 结束时间 | 日期时间选择器 | 绝对时间 |
| 备注 | 文本输入（可选） | - |

**按钮**: 取消 / 确定

**应用逻辑**: 标签自动应用到所有已选版本，用户无需关心

### 3.3 标签存储

- 标签数据存储在数据库中
- 每个标签关联到对比会话（通过 compare_id 或 version_ids）
- 支持跨版本共享标签

## 4. API 设计

### 4.1 版本列表 API

**现有 API**: `GET /api/core/performance-monitor/version/list?device_id={device_id}`

**修改**: 移除 device_id 参数限制，返回所有版本

```typescript
export async function getVersions() {
  return requestClient.get<{ items: PerformanceVersion[] }>(
    '/api/core/performance-monitor/version/list'
  );
}
```

### 4.2 标签 API

**新增 API**: 标签管理（对比页面专用）

```typescript
// 创建标签
export async function createCompareTag(params: {
  name: string;
  type: 'peak' | 'stable';  // 冲高 / 稳态
  start_time: string;       // ISO 格式绝对时间
  end_time: string;         // ISO 格式绝对时间
  note?: string;
}) {
  return requestClient.post('/api/core/performance-monitor/compare/tag', params);
}

// 获取标签列表
export async function getCompareTags(compareId: string) {
  return requestClient.get(`/api/core/performance-monitor/compare/${compareId}/tags`);
}

// 删除标签
export async function deleteCompareTag(tagId: string) {
  return requestClient.delete(`/api/core/performance-monitor/compare/tag/${tagId}`);
}
```

## 6. 实现要点

### 6.1 版本选择逻辑

- 移除 device_id 参数依赖
- 使用所有版本的列表供用户选择
- 版本卡片显示版本名称和颜色标识

### 6.2 时间对齐

- 各版本使用相对时间（从各版本采集开始时间为 0）
- 绝对时间用于标签定义和显示

### 6.3 标签管理

- 标签存储在数据库，关联对比会话
- 标签自动应用到所有已选版本
- 数据摘要统计所有标签合并区间

### 6.4 悬停联动

- 鼠标悬停图表时，各版本进程详情面板同步更新
- MiniTooltip 显示各版本当前时间点数据

## 7. 成功标准

- 版本对比页面与性能监控页面风格统一
- 移除设备选择后，所有标记版本都能正常选择
- 支持更多 HWiNFO 指标对比
- 标签功能简化，用户体验良好
- 数据摘要清晰展示最优/最差值
